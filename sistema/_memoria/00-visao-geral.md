# 00 — Visão Geral do PIPE OS

## O que é o PIPE OS

PIPE OS é o sistema operacional jurídico-comercial da Vieira da Silva Advocacia.

Não é apenas um software jurídico. É a infraestrutura que une operação jurídica, comercial, relacionamento e inteligência de negócio em um único lugar.

O nome PIPE vem de pipeline — o fluxo completo de uma oportunidade desde o primeiro contato até a entrega jurídica e o pagamento.

## Missão

Dar ao escritório a capacidade de crescer com previsibilidade — sabendo de onde vêm os clientes, quanto cada um gera, o que está em andamento, o que está atrasado e o que precisa de ação agora.

## Visão

Referência: G4OS (G4 Educação) — um sistema mãe com subsistemas conectados que operam de forma integrada.

O PIPE OS deve ser o "sistema nervoso central" do escritório imobiliário:
- **Comercial:** imobiliárias, corretores, clientes, oportunidades, propostas
- **Jurídico:** demandas extrajudiciais, consultorias mensais, processos judiciais
- **Gestão:** tarefas, prazos, documentos, relacionamento
- **Inteligência:** dashboard executivo, alertas, IA advisor (futura)

## Escopo atual

O PIPE OS está sendo construído para a **área imobiliária** do escritório:
- ~34 imobiliárias parceiras (contas estratégicas)
- ~500 corretores vinculados (canal de aquisição)
- Clientes pessoa física indicados pelos corretores
- Demandas jurídicas geradas por essa operação

## Premissa central

Todo corretor é um canal de aquisição.
Todo cliente indicado é uma oportunidade.
Toda oportunidade precisa de pipeline.
Todo pipeline precisa de follow-up.
Todo follow-up precisa de sistema.
